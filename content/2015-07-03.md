### [GraphQL](http://facebook.github.io/graphql/#sec-Overview)

a query language created by Facebook in 2012 for describing the capabilities and requirements of data models for client‚Äêserver applications. The development of this standard started in 2015.

**Example**:

-	query:

```
{
  user(id: 3500401) {
    id,
    name,
    isViewerFriend,
    profilePicture(size: 50)  {
      uri,
      width,
      height
    }
  }
}
```

-	result:

```json
{
  "user" : {
    "id": 3500401,
    "name": "Jing Chen",
    "isViewerFriend": true,
    "profilePicture": {
      "uri": "http://someurl.cdn/pic.jpg",
      "width": 50,
      "height": 50
    }
  }
}
```

### [sjsp](https://github.com/itchyny/sjsp)

This is a JavaScript profiler, injecting profiling codes into your JavaScript files.

**example**:

```js
function test() {
  var start_time = Date.now(); // grab the current time at the top

  // our code

  log_profile("test", Date.now() - start_time); // grab the current time again and log the time the function consumed.
}
```
